Create, manage, and organize all your automated tests in one place. Supercheck supports browser, API, database, custom, and performance tests.

## Test Types

**Browser Tests** - Automate web application interactions using Playwright
- User flows, form submissions, navigation
- Visual testing, screenshots
- Cross-browser support

**API Tests** - Validate REST APIs and GraphQL endpoints
- HTTP requests with assertions
- Response validation
- Authentication testing

**Database Tests** - Query and validate database state
- SQL queries
- Data validation
- Schema checks

**Custom Tests** - Write flexible TypeScript/JavaScript tests
- Any testing scenario
- Custom logic and assertions
- Third-party integrations

**Performance Tests** - Load testing with k6
- Stress testing
- Scalability validation
- Performance benchmarking

## Example: Browser Test

```typescript
// Navigate and interact
await page.goto("https://example.com/login");
await page.fill("#email", "user@example.com");
await page.fill("#password", getSecret("PASSWORD"));
await page.click("button[type='submit']");

// Assert success
await expect(page.locator(".dashboard")).toBeVisible();
```

## Example: API Test

```typescript
// Make API request
const response = await fetch("https://api.example.com/users", {
  headers: { Authorization: `Bearer ${getSecret("API_KEY")}` }
});

// Validate response
expect(response.status).toBe(200);
const data = await response.json();
expect(data.users).toHaveLength(10);
```
