## Features

- **Playground** — Write and test with live validation
- **AI Fixes** — Automatic suggestions when tests fail
- **Rich Reports** — Screenshots, traces, and metrics
- **Scheduling** — Cron-based with timezone support
- **Multi-Region** — Run performance tests from US, EU, APAC
