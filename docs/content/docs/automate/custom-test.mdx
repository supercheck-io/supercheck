# Custom Test

Write fully custom test scripts for advanced testing scenarios.

## What is a Custom Test?

Custom tests give you complete flexibility to:
- Combine multiple test types
- Use custom libraries
- Implement complex logic
- Integrate with external services

3. Use any npm packages you need
4. Run and debug your test

## Example Test

```typescript
import { test, expect } from '@supercheck/test';
import axios from 'axios';

test('complex workflow test', async () => {
  // Step 1: Create user via API
  const createResponse = await axios.post(
    'https://api.example.com/users',
    { name: 'Test User', email: 'test@example.com' }
  );

  const userId = createResponse.data.id;

  // Step 2: Verify user exists
  const getResponse = await axios.get(
    `https://api.example.com/users/${userId}`
  );

  expect(getResponse.data.name).toBe('Test User');

  // Step 3: Delete user
  await axios.delete(`https://api.example.com/users/${userId}`);
});
```
