## Schedule Options

| Type | Description |
|------|-------------|
| **Interval** | Every X minutes/hours |
| **Cron** | Custom cron expression |
| **Manual** | Run on demand |
| **API Trigger** | Remote execution via API |

## CI/CD Integration

Trigger jobs remotely from your CI/CD pipeline.

### Generate API Key

1. Open job settings
2. Go to **CI/CD** tab
3. Click **Generate API Key**

![CI/CD Tab](/screenshots/edit-job-add-cicd-apikey-1.png)

![Generate API Key](/screenshots/edit-job-add-cicd-apikey-2.png)

### Configure Trigger

Copy the trigger URL and API key for your CI/CD pipeline.

![API Key Configuration](/screenshots/edit-job-cicd-apikey-configure.png)

### Example Request

```bash
curl -X POST https://your-instance.com/api/jobs/{jobId}/trigger \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Best Practices

- Group related tests into a single job
- Set appropriate failure thresholds
- Configure alerts for production jobs
- Use cron for specific timing requirements
- Generate unique API keys per CI/CD pipeline
