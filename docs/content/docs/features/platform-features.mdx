# Core Capabilities

Supercheck combines testing, monitoring, and intelligent automation so your team can trust every release. Explore the key areas below to see how everything fits together.

## Product Pillars

<Tabs items={['Test', 'Monitor', 'Collaborate', 'Improve']}>
  <Tab value="Test">
    Design browser, API, and custom checks in one place with reusable snippets,
    shared variables, and rich reports.
  </Tab>
  <Tab value="Monitor">
    Keep customer journeys and critical endpoints under watch with smart
    alerting and easy-to-share status pages.
  </Tab>
  <Tab value="Collaborate">
    Assign roles, leave run notes, and notify the right people instantly across
    jobs, monitors, and incidents.
  </Tab>
  <Tab value="Improve">
    Use insights and AI-powered fix suggestions to shorten feedback loops and
    catch regressions before users do.
  </Tab>
</Tabs>

## Always-On Monitoring

| What you monitor | How it helps | Extras |
| ---------------- | ------------ | ------ |
| **Synthetic journeys** | Validate full customer flows every few minutes | Compare runs side-by-side |
| **APIs & web endpoints** | Track uptime, latency, and response shapes | Add headers and auth securely |
| **Public status pages** | Share real-time health with stakeholders | Auto-aggregate incidents |

<Steps>
  <Step>Choose what to monitor: journey, endpoint, or data check</Step>
  <Step>Set frequency and alert rules that match your SLA</Step>
  <Step>Select notification channels — email, Slack, or webhook</Step>
  <Step>Publish optional status pages for customers or partners</Step>
