# Team Collaboration & Security

Supercheck is built for teams. This guide shows how to invite colleagues, assign roles, and protect sensitive data without slowing anyone down.

## Invite Your Team

<Callout type="tip">
  New teammates land in your default project. Create additional projects to keep
  departments or products separated.
</Callout>

## Roles at a Glance

| Role | What they can do | Ideal for |
| ---- | ---------------- | --------- |
| **Owner** | Manage billing, security, and organization-wide settings | Platform admins |
| **Admin** | Configure projects, monitors, and notifications | Reliability leads |
| **Editor** | Create and run tests, jobs, and monitors | Engineers, QA analysts |
| **Viewer** | Review dashboards, runs, and status pages | Stakeholders, support |

## Keep Data Safe

- Secrets are encrypted automatically; never paste sensitive values into descriptions or notes.
- Set up single sign-on (SSO) from the settings page if your company uses an identity provider.
- Turn on two-factor authentication for added protection.
- Review the audit log to see who changed what and when.

## Handling Incidents Together

<Tabs items={['Notify', 'Coordinate', 'Review']}>
  <Tab value="Notify">
    Use targeted alert channels so the right people respond quickly without
    spamming the entire company.
  </Tab>
  <Tab value="Coordinate">
    Add run notes, attach links, and tag teammates to keep investigation history
    in one place.
  </Tab>
  <Tab value="Review">
    After resolution, export the incident timeline and share highlights in
    retrospectives.
  </Tab>
</Tabs>

## Compliance & Requests

- Need a security questionnaire or compliance pack? Email security@supercheck.io.
- Data residency options and retention policies are available for enterprise plans.
- Request access reviews quarterly to keep your user list accurate.

<Callout type="note">
  Security is a shared responsibility. If you spot anything unusual, reach out
  to the Supercheck security team immediately.
</Callout>
