# Communicate

The Communicate section provides tools to share system status, manage incidents, and keep your users informed about the health of your services.

## Overview

Supercheck's communication features help you:

- **Monitor Health**: Get real-time insights into your system's health with comprehensive dashboards
- **Respond to Incidents**: Set up intelligent alerting to catch issues before they impact users
- **Build Trust**: Create beautiful, public-facing status pages to keep stakeholders informed

### Monitoring System Health

The Dashboard provides a centralized view of all your:

- **Active Tests**: Track test execution trends and success rates
- **Job Runs**: Monitor automated job executions
- **Monitor Status**: Real-time health of all monitoring endpoints
- **System Capacity**: Track queue and execution capacity
