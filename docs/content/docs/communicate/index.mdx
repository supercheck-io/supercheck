## Threshold Logic

Prevent alert noise with consecutive failure thresholds:

| Check | Status | Result |
|-------|--------|--------|
| 1 | ❌ Fail | No alert (1/3) |
| 2 | ❌ Fail | No alert (2/3) |
| 3 | ❌ Fail | **Alert sent** |
| 4 | ✅ Pass | Recovery sent |

## Status Pages

Create public status pages with:

- **Custom branding** — Logo and colors
- **Component status** — Operational, degraded, outage
- **Incident history** — Timeline of issues
- **Scheduled maintenance** — Planned downtime
- **Custom domains** — Your subdomain
