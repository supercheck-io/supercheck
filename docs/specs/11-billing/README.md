# Billing & Monetization Documentation

This section covers billing integration, payment processing, and usage-based pricing for SuperCheck.

## Files

- **[POLAR_BILLING_INTEGRATION.md](POLAR_BILLING_INTEGRATION.md)** - Complete guide for Polar.sh payment platform integration including setup, pricing tiers, usage-based billing, and hard stop limits

## Billing Overview

- **Payment Provider**: Polar.sh
- **Pricing Model**: Subscription-based with usage limits
- **Billing for**:
  - Playwright execution minutes
  - K6 load testing VU minutes
  - Plan-based limits for monitors, status pages, projects, and team members
- **Self-Hosted**: Always free and unlimited
- **SaaS**: Subscription tiers with different feature access
